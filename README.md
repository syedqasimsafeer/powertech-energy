# PowerTech Energy Solutions Website

This is a beginner-friendly static website for GitHub Pages.

## Files
- `index.html` = website structure and content
- `style.css` = design, colors, layout, mobile view
- `script.js` = mobile menu and inquiry email form
- `assets/` = logo and image placeholders

## How to edit
1. Open `index.html` in VS Code.
2. Change text, phone numbers, services, or address.
3. Replace images inside `assets/`.
4. Upload/commit the updated files to GitHub.
5. GitHub Pages will update your website automatically after a short time.

## Recommended real image names
Use these file names to avoid editing code:
- `assets/logo-placeholder.svg` can be replaced by `assets/logo.png` if you also update the HTML.
- `assets/circuit-placeholder.svg`
- `assets/lab-placeholder.svg`

If you upload your real images later, update the `src=""` paths in `index.html`.
